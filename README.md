# mon_amour

Mon Amour — мобильное приложение магазина одежды на Kotlin и Jetpack Compose. 

###  Реализованный функционал:
* Авторизация пользователей.
* Каталог товаров с изображениями.
* Корзина покупок.
* Сохранение данных через SharedPreferences.
* Раздел товаров со скидками.
